# **App Name**: Global Conflict Compass

## Core Features:

- ACLED Conflict Map: Displays conflict event data from ACLED (Armed Conflict Location & Event Data Project).
- ReliefWeb Humanitarian Updates: Presents reports and analysis related to humanitarian relief efforts from ReliefWeb.
- BBC Conflict News Feed: Aggregates and displays news articles from BBC related to global conflicts, filtering for relevance.
- Global Peace Index Ranking: Shows the latest rankings from the Global Peace Index (GPI), highlighting the most and least peaceful nations.
- AI Conflict Summary: Summarizes the titles and descriptions of news from different sources with a generative AI tool. The AI is able to include or exclude the link for reading more, based on the summarized content.

## Style Guidelines:

- Primary color: A muted blue (#5DADE2) evoking a sense of calm and information.
- Background color: Light gray (#F0F4F7) for a clean, readable backdrop that doesn't distract from the data.
- Accent color: A contrasting orange (#E3A253) to highlight key information and interactive elements.
- Clean, sans-serif font for readability.
- Simple, neutral icons for categories like location, date, and source.
- Card-based layout to clearly separate information from different sources, arranged in a responsive grid.
